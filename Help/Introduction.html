Gurux GXDLMS/COSEMDirector

Introduction

GXDLMS/COSEMDirector

Gurux GXDLMS/COSEMDirector is an easy-to-use application to communicate with DLMS devices.

You can connect to one or more physical DLMS devices
with GXDLMS/COSEMDirector and you can select whether to read values from all listed devices, or one of them, or
just a single register of a single device.


    Note: the physical device may get disconnected,
    if it is busy doing its main task, which is measuring
    consumption of electricity.



GXDLMS/COSEMDirector supports:


Operating systems:
  • Windows 7
  • Windows Vista
  • Windows 2000
  • Windows XP
 
Connection types
  • Net
  • Serial
  • Terminal



IEC standards
  • IEC 62056-21 Direct local data exchange
  • IEC 62056-42 Physical Layer Services and Procedures for Connection-Oriented Asynchronous Data Exchange
  • IEC 62056-46 Data link layer using HDLC protocol
  • IEC 62056-47 COSEM transport layers for IPv4 networks
  • IEC 62056-53 COSEM application layer
  • IEC 62056-61 OBIS Object identification system
  • IEC 62056-62 Interface objects

System requirements:

  • .NetFramework 2.0 or newer needed
  • 1 GHz processor
  • 1 GB memory
  • 300 MB free hard disk space
