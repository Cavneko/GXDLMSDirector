Gurux GXDirector

GXDirector Settings

Device Tree

Determines which items are shown in Device Tree and whether Context Menu is available by right-clicking.

Event Log

Determines maximum amount of rows shown in the Event Log tab.

GXDirector Settings

Determines if device templates are loaded automatically when a device list is loaded.

You can also show or hide the Scheduled Tasks tab in the tab control.

Database settings

To write data to a database, add a DataLogger and set its properties.

See Writing to a Database.

File settings

You can add or remove log files and set their properties.

Note: ClearSize and MaxSize are entered as row counts, 0 = None.
