File menu

In addition to normal file operations File menu includes Find Device command (sub command for New) for searching physical devices in your network.

File menu also includes Open and Save commands for Value Lists. Read more about managing Value Lists.

View menu

In View menu you can select items shown in Device Tree.

View as Hex command shows Property values as hexadecimal.

Find command searches items branching from the selected item in the Device Tree.

For more information, see Help on Views.

Tools

Tools menu has commands for transactions between the computer and physical device.

In addition Tools menu has Properties command for opening properties of the selected item for editing. When connected to a device, Device properties can be viewed but not edited.

Schedule Settings, see Scheduled Tasks help.

GXDirector Settings, see GXDirector Settings help.

Macro, see Macros help.

With Language command you can switch languages instantly without restarting GXDirector.

Help menu

Library Versions command shows version numbers of GXDirector and libraries. Gurux technical support may ask you this information in case of troubleshooting.

Tip: Commonly used commands can also be accessed in the context menu (right-click) or with buttons on the toolbar.

Content views of items selected in the Device Tree

If no UI has been created and attached to the device or to any of its items, various data is shown in the content view, according to the item selected in the Device Tree.

Selecting a device list in the Device Tree shows a list of all devices in the content view. Listed are the name, group, type and status of the devices. Selecting a device group shows a similar list of devices in that particular group.

Selecting a device shows all categories and tables in the device as tabs. Category tabs show the property name, value, data type, unit, time of last transaction, read and write count and execution time. Read and Write Count columns show amount of transactions / errors. Execution Time column shows duration of last transaction / average duration in milliseconds. Table tabs show the table columns.

Selecting a category shows a list of its properties in the content view. Listed are property name, value, data type, unit, time of last transaction, read and write count and execution time. Read and Write Count columns show amount of transactions / errors. Execution Time column shows duration of last transaction / average duration in milliseconds.
Property values can be read, edited and written by commands in context menu (right-click). They can also be reset to default or last read value. Selecting a table in Device Tree shows the table columns in the content view.

Selecting a single property in the Device Tree shows general information of the property in text fields. Shown are Name, Type, Unit, Timestamp and Value of the selected property. If the AccessMode of the property is set to Write or ReadWrite in GXDevice Editor, the value of the property is editable and the field turns to white, when the device is connected. If the AccessMode of the property is set to NotAccessible or Read, the value cannot be edited and the field stays grayed.
